# Servlet Setup with Age Calculator (Tomcat Demo)

A simple **Java Servlet web application** used to verify the installation and configuration of **Apache Tomcat**.
The project calculates a user's age based on the entered date of birth and confirms that the **Java Web Environment is working correctly**.

This project was created as a **test deployment application** before moving on to larger Java web projects.

---

## Purpose

The goal of this project is to confirm that:

* Apache Tomcat is installed correctly
* Java Servlets are configured properly
* WAR deployment works
* A browser can communicate with the servlet container
* The Java web development workflow is functioning

---

## Features

* Simple **Age Calculator**
* Built using **Java Servlet**
* Runs on **Apache Tomcat**
* HTML form input
* Dynamic response generated by the servlet
* Lightweight verification project for Java web setup

## How It Works

1. The user enters their **Date of Birth** in the HTML form.
2. The form sends a request to the **Servlet**.
3. The servlet calculates the age using Java.
4. The calculated age is returned and displayed in the browser.

---

## Running the Project

### 1. Install Requirements

* Java JDK
* Apache Tomcat

---

### 2. Start Tomcat

Navigate to the Tomcat `bin` directory and start the server.

Windows:

```bash
startup.bat
```

### 3. Deploy the Application

Copy the compiled **WAR file** into the Tomcat `webapps` folder.

Example:

```text
Tomcat/webapps/AgeCalculator.war
```

Tomcat will automatically deploy the application.

---

### 4. Open in Browser

```text
http://localhost:8080/AgeCalculator
```

Enter your **Date of Birth** and the servlet will calculate your age.
